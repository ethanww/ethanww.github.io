python小练习(3):将优惠券保存到MySQL中

题目：

将上一题生成的200个优惠券保存到MySQL关系型数据库中

说明

这里使用了PyMySQL，这是Python3.x(博主的环境是python3.5)版本中用于连接MySQL服务器的一个库。如果你使用的是2.x的版本，那么可以使用mysqldb。

步骤

首先，需要安装PyMySQL，建议使用pip，方便快捷：)

然后安装mysql，网上资源很多，这里篇幅有限，不再赘述。安装完成后，写一个coupon.sql文件创建数据库和表

然后执行：
mysql -u root -p <coupon.sql

没有出现问题表示创建成功。
然后，我们开始写代码

运行后可以查看结果
